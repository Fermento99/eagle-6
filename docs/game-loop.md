# Game loop

## Preparation

1. Assign players
   - random
   - manual
2. Randomize code names
3. Pick starting team

## Main Loop

1. Pick Encryptor
   - from a cycle
   - by submission
2. Randomize code
3. Await for clues
4. Await for teams' submissions
5. Reveal code and submissions
6. Award points
7. Check for winning condition
8. Change the leading team
9. Repeat from 4.
